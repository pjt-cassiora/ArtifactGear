# アーティファクトギア バトルシミュレータ

プロジェクトカシオレによるカードゲーム「アーティファクトギア」のゲームバランスを検証するためのPythonシミュレータです。
2026-04-19

## 機能

- **ソロプレイモード（人間プレイ）**: 人間が思考する順番（特技確認 → 使い捨てスキル確認 → ターゲット＆攻撃方法選択）に沿ってプレイできるモードです。自分がモンスターを討伐した際に獲得ギアの取捨選択を行え、装備戦略のコントロールが可能です（NPC討伐時は自動装備）。
- **ラスボス決戦（2連戦）**: モンスター10体討伐後に移行するクライマックス。代表者1名を選出し、パーティ全員のサポートで邪悪なドラゴンに挑みます。
- **共有リソース（Shared Pool）**: 第1戦開始時にパーティ全員の成功カードが共有プールへ集約されます。代表者はこの巨大なリソースを背景に戦います。第2戦ではボスの特殊能力「手札破壊」を生き残ったカードが共有資産として場に蓄積されます。
- **詳細な統計分析**: 高速シミュレータにより、初戦突破率・再戦突破率の内訳、代表アタッカーの職業分布まで詳細に分析可能です。
- **パーティ編成網羅分析**: 重複なしの職業組み合わせ（全25通り）を網羅テストし、詰み編成がないか検証。
- **共通デッキと両面カード**: 成功と失敗を「両面」として統合した36枚の共通デッキによるリソース管理。
- **職業別「特技」システム**: 各職業固有の1回限定の切り札。召喚士はボス戦において自分を含む全員の戦闘に割り込み可能です。

## 実行方法

### ラスボス決戦（フルプレイ：雑魚10体 + ボス2連戦）
```bash
./run-cmd.bat python 04_boss_battle_play_cui.py
```

### ボス戦シミュレート（バランス検証）
```bash
./run-cmd.bat python 03_boss_battle_sim.py
```

### パーティ勝率分析（網羅テスト）
職業の重複なしルールに基づき、2〜4人プレイの全25通りを網羅的に分析します。
```bash
./run-cmd.bat python 05_party_analyze_sim.py
```

### 道中ザコ戦のみ
```bash
./run-cmd.bat python 02_monster_battle_play_cui.py   # 対話プレイ
./run-cmd.bat python 01_monster_battle_sim.py         # シミュレーション
```

**注意**: `run-cmd.bat` 環境では標準入力がフリーズするため、対話型スクリプトの直接テストは非推奨です。テストは `pytest` で実行してください。

## 設定

`setting.py` で以下の設定を変更できます：

### 基本設定

| 設定名 | 現在値 | 説明 |
|--------|-----------|------|
| `GEAR_SLOT_LIMIT` | 3 | パーティ共有ギアの装備枠数 |
| `SIMULATION_COUNT` | 10000 | シミュレーション回数 |
| `PLAYER_COUNT` | 4 | プレイヤー人数（2〜4） |
| `MONSTER_COUNT` | 10 | 1ゲームに出現するモンスターの総数（最大20） |
| `FIELD_SIZE` | 3 | 場のモンスター補充数 |
| `TOTAL_MONSTER_COUNT` | 20 | 全モンスター種類数（CSVの行数） |
| `CARD_TOTAL` | 36 | 両面カード総枚数（成功面・失敗面を1枚と数える） |
| `SKILL_USE_PROBABILITY` | 0.8 | 使い捨てスキルを使用する確率 |
| `UNIQUE_SKILL_USE_PROBABILITY` | 0.6 | 特技を使用する確率 |
| `DEFAULT_MODE` | 'sim' | デフォルトモード（'log' または 'sim'） |
| `HAND_LIMIT` | 26 | 手札上限枚数 |
| `SUMMONER_MAX_BONUS` | 3 | 召喚士の精霊王召喚による最大追加攻撃力 |

### ボス戦設定

| 設定名 | 現在値 | 説明 |
|--------|-----------|------|
| `BOSS_PHASE1_TARGET` | 17 | 第1戦（邪悪なドラゴン）の目標値 |
| `BOSS_PHASE2_TARGET` | 19 | 第2戦（怒りの邪悪なドラゴン）の目標値 |

## ゲームルール概要

### 道中の試練（ザコ戦フェーズ）
1. **初期化**: 山札から10体のモンスターをシャッフル。各プレイヤーにランダムな職業を割り当て。
2. **ターン処理**: 場のモンスター（最大3体）から最も勝率の高い敵を選択し、1d6＋ステータスで討伐判定。
3. **成功時**: ギア獲得（パーティ共有3枠）＋成功カード1枚ドロー。
4. **失敗時**: 失敗カード1枚ドロー。撤退効果があればモンスターを場から除外。「再戦可能」効果の場合はモンスターを場に残す。
5. **クリア条件**: 全10体を討伐/撤退して場から排除。

### ラスボス決戦（ボス戦フェーズ）
1. **リソース集約**: 第1戦開始時に、全員の手札にある成功カードをすべて「共有プール」へ集約。
2. **第1戦（邪悪なドラゴン・オモテ）**: 目標値17。代表者1名のステータスに共有プールの全ボーナスを加算して判定。勝利で全員成功カード1枚獲得。
3. **手札破壊**: 第2戦の**最初の一回のみ**、プールと報酬カードを合算した中からプレイヤー人数分（4枚）を没収。
4. **第2戦（怒りの邪悪なドラゴン・ウラ）**: 目標値19。没収を免れた全カードを「装備」として、代表者が最終決戦に挑む。
5. **再挑戦**: 各フェーズにつき最大2回のリトライが可能（フェーズ移行時にカウンタはリセット）。敗北時はパーティ全体で4枚の失敗カードを受け取り、内訳を表示。最大回数に達した場合は敗退。

## 🌟 職業別特技（Unique Skill System）

| 職業 | 基礎ステータス (武/魔) | 特技名 | 効果 |
|------|-----------|--------|------|
| 剣士 | 4 / 0 | 乾坤一擲 | サイコロ変換（1→5、それ以外→10） |
| 魔法剣士 | 3 / 1 | 幻影剣術 | 3回振って最大の出目に+2 |
| 僧侶 | 2 / 2 | 祝福の祈り | 捨て札・山札から各1枚成功カードを取得する。即時使用可 |
| 召喚士 | 1 / 3 | 精霊王召喚 | サイコロ振ったあとに加護を付与（奇数:+3、偶数:+2）。自分を含め、どのプレイヤー時でも適用可。 |
| 魔法使い | 0 / 4 | 魔力暴走 | 出目5以上で12扱い、それ以外で成功カード1枚獲得 |

## 注意事項

- 道中で「成功カード」をどれだけ温存できるかがボス戦の鍵です。
- 第2戦の「手札破壊」で共有プールに残るカードの質と量が、最終局面の勝敗を左右します。
- スキル使用: プレイヤーは戦闘前に、自分や仲間の手札、あるいは共有プールにある「使い捨てスキル」を自由に使用できます。使用したカードのみが捨て札となります。
